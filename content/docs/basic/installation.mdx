## 前置条件

在安装 REAREye 之前，请确保你的设备满足以下条件：

1. 已 Root 的 小米 17 Pro 或 Pro Max 设备
2. 已安装 **LSPosed** 框架

## 安装步骤

### 1. 下载模块

| **版本** | 稳定版 | 测试版 |
| ---- | ---- | ---- |
| **特点** | 稳定版经过多轮测试的校验，<br />使用过程中不会遇到什么问题，<br />但是不会拥有最新的功能特性 | 测试版包含最新功能，<br />但可能未经过充分测试<br /><u>**下载需要登录 GitHub 账号**</u> |
| **下载** | [GitHub Releases](https://github.com/killerprojecte/REAREye/releases) | [Github Actions](https://github.com/killerprojecte/REAREye/actions) |

或加入 [**QQ群**](https://qm.qq.com/q/h7e2si9R74) 从群文件中下载最新版本。
### 2. 安装

从上面的途径获取到 APK 文件后

通过文件管理器或包管理器在您的设备中安装

### 3. 在 LSPosed 中激活

1. 打开 **LSPosed** 管理器
2. 找到 **REAREye** 模块
3. 勾选启用，并选择作用域（需要Hook的目标应用）
4. 重启设备或目标应用

<Callout type="warn">
  修改作用域后需要重启目标应用才能生效。
  **如果是首次安装请重启您的设备**
</Callout>
