### 系统框架

1. **活动白名单**: 
    - 模式: 白名单 / 全选
    - 作用: 允许选中的应用在背屏中启动活动
2. **后台白名单**:
    - 白名单应用：允许被选中的应用不受系统限制在后台运行
    - 锁定应用：以系统用户的身份锁定选中的应用
3. **阻止锁屏后背屏返回桌面**: 
    - 开启后锁屏将不会强制让在背屏运行的应用返回桌面
4. **禁用背屏保护**:
    - 开启后主屏全屏播放视频时将不会锁定背屏


### 背屏中心

## 背屏动态通知显示

在 `1.0.4-40f1c1b-r65-canary` 及以后的版本里，新增了**背屏动态通知组件显示**的功能。

基于此功能与组件商店（*本功能将在未来版本中提供*）中的相关组件，可以做到 **HyperOS 焦点通知（超级岛）** 在背屏的显示。

# 具体使用方法

1. **进入设置**：打开“背屏中心”，进入“背屏个性化”。
2. **管理组件**：在背屏个性化页面，选择“**组件模板管理器**”，点击“新增组件”，选择相关背屏组件。
3. **填写名称（关键）**：
   > **[最关键的一步]** 务必将组件名称填写为组件开发者需要的名称。
   > *例如：背屏课程表 Pro 的组件名称为 `course_schedule`。*
4. **开启权限**：回到背屏个性化界面，打开“**允许焦点通知在背屏显示**”。
5. **生效设置**：返回 REAREye 的主页，点击右上角并选择“**快捷终止背屏中心**”或直接**重启手机**。

---

**预期效果：**
如果不出意外，背屏动态通知组件将会在主屏动态通知（超级岛）显示时，同时在背屏显示。
