### 系统框架

1. **活动白名单**: 
    - 模式: 白名单 / 全选
    - 作用: 允许选中的应用在背屏中启动活动
2. **后台白名单**:
    - 白名单应用：允许被选中的应用不受系统限制在后台运行
    - 锁定应用：以系统用户的身份锁定选中的应用
3. **阻止锁屏后背屏返回桌面**: 
    - 开启后锁屏将不会强制让在背屏运行的应用返回桌面
4. **禁用背屏保护**:
    - 开启后主屏全屏播放视频时将不会锁定背屏
